
HOW TO PUBLISH YOUR RDA CHOOSEN WEBSITE

1. Create a GitHub account and new repository (public).
2. Upload all files from this ZIP into the repository.
3. Go to: Settings > Pages > Branch = "main" > Folder = "/root".
4. Save. Your site will go live in 1 minute at: https://<username>.github.io/<repo-name>/

To customize:
- Replace product images and links inside index.html
- Update logo in /img/
- Add WhatsApp or payment links to Buy buttons
- Optional: Connect your custom domain
